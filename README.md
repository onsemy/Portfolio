### [JJ4Unity](https://github.com/onsemy/JJ4Unity), [JJ4UnityVSC](https://github.com/onsemy/JJ4UnityVSC)

- Unity 개발에 도움이 되는 코드 모음
- 주요 기능
  - Addressables 암/복호화 기능
    - 주요 코드 위치: [[EncryptedBuildScript.cs](https://github.com/onsemy/JJ4Unity/blob/main/Assets/JJ4Unity/Editor/AssetBundle/EncryptedBuildScript.cs)] [[EncryptedAssetBundleProvider.cs](https://github.com/onsemy/JJ4Unity/blob/main/Assets/JJ4Unity/Runtime/AssetBundle/EncryptedAssetBundleProvider.cs)]
  - `public` 또는 `[SerializeField]`로 선언된 멤버 변수를 자동으로 할당해주는 AssignPath 기능
    - 주요 코드 위치: [[AssignPath.cs](https://github.com/onsemy/JJ4Unity/blob/main/Assets/JJ4Unity/Runtime/Extension/AssignPath.cs)]
  - VSCode -> Unity Editor 에셋 갱신 기능
    - 주요 코드 위치: [[JJ4UnityInitializer.cs](https://github.com/onsemy/JJ4Unity/blob/main/Assets/JJ4Unity/Editor/JJ4UnityInitializer.cs)] [[extension.ts](https://github.com/onsemy/JJ4UnityVSC/blob/master/src/extension.ts)]
- 사용 기술: Unity, C#, Addressables, Typescript

### [PyExcelToMustache](https://github.com/onsemy/PyExcelToMustache), [PyExcelToBson](https://github.com/onsemy/PyExcelToBSON)

- Excel 파일을 코드 템플릿([Mustache](https://github.com/onsemy/PyExcelToMustache))이나 Binay JSON([BSON](https://github.com/onsemy/PyExcelToBSON))으로 변환하는 툴
- 사용 기술: Python
